# WeatherApp

This is an initial development to achieve a responsive weather app.

Language : Kotlin

Support : Android Lollipop (API 21 onwards)

Architecture includes :
  1. MVVM
  2. LiveData
  3. AndroidX
  4. Jetpack
  5. Coroutines
  6. Retrofit
  7. GSON
  8. Room Database
  9. Google Play Services
  10. Constraint Layout
  11. Lifecyle Aware
  12. Glide
